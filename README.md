# Imager Tiler

## Quick Start

Add an image of your chosing to the root directory of the application e.g. Cat.jpg.

````
git clone https://github.com/keithbro/image-tiler.git
cd image-tiler
npm install
npm start Cat.jpg
````

## Prerequisites

Node v12 or higher.

## Tests

Test have been created under `tests/`. To run:

````
npm test
````
